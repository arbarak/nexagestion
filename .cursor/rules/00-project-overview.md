# 00 – Project Overview (NexaGestion)

## What this project is

NexaGestion is a **multi-sociétés ERP de gestion commerciale** pour PME marocaines, avec un focus particulier sur :

- Multi-sociétés (groupe de 4 sociétés)
- Données de base partagées au niveau groupe :
  - Clients, fournisseurs, articles, services, familles, marques, taux de TVA/TSP
- Opérations isolées par société :
  - Ventes, achats, stock, paiements, journaux, rapports
- Gestion de stock avancée
- Module maritime :
  - Bateaux, interventions, sorties/retours d’articles
- Suivi des employés :
  - Actions, temps de travail, visites, interventions bateau
- Ventes :
  - Devis, commandes, BL, factures, avoirs, proforma
- Rapports avancés (ventes, achats, stock, trésorerie, bateaux, employés, TVA)
- Conformité Maroc (ICE, IF, RC, Patente, TVA/TSP, journaux comptables)

## What to optimize for

- **Performance & réactivité** (app légère, rapide)
- **Lisibilité & maintenabilité** (architecture claire)
- **Simplicité pour l’utilisateur métier** (pas un jouet de développeur)
- **Respect strict des règles métier Maroc / multi-sociétés**
